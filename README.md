Hoje apresento o site da modelo fictícia Anna Bela.
É um projeto simples, desenvolvido como parte do meu processo de aprendizado em desenvolvimento web.
A proposta foi criar uma página organizada, com identidade visual coerente e uma apresentação profissional da personagem, aplicando conceitos básicos de estruturação em HTML e estilização com CSS
Além da construção do conteúdo
trabalhei a configuração em CSS para definir cores, fontes, espaçamentos e alinhamentos
buscando deixar o visual mais agradável e estruturado. Mesmo sendo algo inicial, já me preocupei com organização de layout e padronização visual.
Reconheço que ainda é um site simples, mas ele representa minha evolução prática e minha dedicação em aprender fazendo
Cada projeto é um passo a mais, e com estudo e consistência, versões cada vez mais completas e profissionais virão
